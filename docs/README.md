# Каталог растений - Сад

Полнофункциональный каталог растений с поиском по названию.

## Описание

Сайт представляет собой статический каталог растений из вашего сада, созданный на основе данных из:
- **Excel файла** (d:\garden\Каталог растений 12.2025.xlsx) — основной список из 612 растений
- **Word документов** (21 файл) — подробные статьи с фотографиями

## Возможности

✅ **Поиск по названию** — ищите растения по русскому или латинскому названию
✅ **Фильтры** — отфильтруйте по зоне зимостойкости и условиям произрастания
✅ **Галерея фотографий** — 1000+ фотографий растений
✅ **Подробные статьи** — полная информация о каждом роде растений
✅ **Быстрая загрузка** — статический сайт, работает без сервера

## Запуск локально

### Способ 1: Встроенный Python сервер

```bash
cd site
python -m http.server 8000
```

Затем откройте в браузере: **http://localhost:8000**

### Способ 2: Развертывание на хостинге

Просто скопируйте содержимое папки `site/` на любой статический хостинг:
- GitHub Pages
- Netlify
- Vercel
- Any-другой хостинг статических сайтов

## Как обновить данные

Если вы измените Excel файл или Word документы:

```bash
python extract.py
```

Этот скрипт:
1. Прочитает все .xlsx и .docx файлы из d:\garden\
2. Извлечет текст и фотографии
3. Создаст JSON файлы в папке `site/data/`
4. Автоматически сопоставит растения из каталога со статьями
